# Project Agent Rules (finance-mvp)

- For every UI/UX/frontend change, read and follow:
  - `skills/balance-design-system/SKILL.md`

- Keep mobile-first behavior and avoid horizontal overflow.

Documentation:
- Update `CHANGELOG.md` on every code-change task.
- Update `BACKLOG.md` when scope or priorities change.
